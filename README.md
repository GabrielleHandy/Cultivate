# Cultivate

AI-powered learning and lifestyle apps.

## Apps

- [`wearit/`](./wearit) — Wardrobe AI. Photograph your clothes, get outfit suggestions powered by Claude + on-device Gemma.
- `cultivate-app/` *(coming soon)* — Personalized AI bootcamp generator. Learn anything through real scenarios and adaptive mentorship.

## Getting Started

Each app is a standalone Expo project. Navigate into the app folder and run:

```bash
cd wearit
npm install
npx expo start
```
